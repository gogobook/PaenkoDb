What is PaenkoDb?

PaenkoDB is a distributed and document-oriented Database which is designed to be fast and reliable.

Motivation

This project is a school project.

Safe

We want our Database to be safe and fast. To achieve this we use the programming language Rust because of the way it handles garbage collection and memory allocation. To learn more about rust visit rust's website

Distributed

To synchronize data between our Nodes we use the up and coming consensus algorithm Raft. Our project is build on top of the implementation by Hoverbear

Document-Oriented

PaenkoDB is document oriented. This means that it can save data schemeless.
